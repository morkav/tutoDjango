Inclusion dans les templates

Les inclusions ne sont PAS le mécanisme principal pour réutiliser des templates en Django. Vous devez utiliser l'héritage avant tout.

Les inclusions sont à utiliser uniquement quand vous avez un petit morceau d'HTML que vous avez besoin de réutiliser à plusieurs endroits, mais pas très souvent.

Cela fonctionne comme on s'y attend :

<div>
    { % include 'app6_included.html' % }
</div>

<div>
    { % include 'app6_included.html' % }
</div>
produira :
{% include 'app6_included.html' %}
{% include 'app6_included.html' %}

ATTENTION : le template inclus et le template incluant ont leurs rendus effectués séparément, ensuite seulement ils sont fusionnés. Cela signifie que vous ne pouvez pas accéder aux variables déclarées dans l'un et l'autre. Si vous en avez besoin, utilisez with :

{ % include 'app6_included.html' with variable=value % }

Les inclusions et l'héritage peuvent être utilisés ensemble sans problème.
